> [Yeoman](http://yeoman.io) generator for AngularJS and Polymer apps. A work in progress.

## Usage

Install `generator-ng-poly`:
```
npm install -g generator-ng-poly
```

Run `yo ng-poly`
Yeoman will then ask for an app name.

Run `gulp` to build.
